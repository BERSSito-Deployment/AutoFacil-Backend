# AutoFacil-Backend

API para simular el credito de un auto con compra inteligente (interbank)

## Compilar

```cmd
cd AutoFacil-Backend-main
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
